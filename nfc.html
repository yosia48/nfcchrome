Absensi & Registrasi Kartu (Web NFC)

Scan kartu → tampilkan data pemilik → simpan absensi / registrasi

Mode: Demo UI (tanpa backend)
Alur cepat: Scan kartu → data pemilik muncul → klik Simpan Absensi.
Jika kamu mau, data pemilik bisa juga ditulis ke NFC tag (NDEF Text/JSON) agar bisa dibaca kembali saat scan.
Kamu bisa generate payload otomatis dari form Registrasi.
Pastikan halaman berjalan di HTTPS atau localhost, dan pakai Chrome Android.
